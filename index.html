Term Project

  I used this project as an opportunity to learn the basics of Blender, and to make an animated version of one of my favorite wallpapers.
The end result (the background on this site!) is a sun object with boolean cutouts that continually loop an animation, creating a vaporwave effect.
Additionally, I used geometry nodes to create all the textures in the scene, most notably on the sun and the space background.
The project can be downloaded as the rendered animation in .webm format, or as the .blend file.
